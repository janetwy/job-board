# Recipe Search App

Main Objective:
	- Allow user to search for recipes
 	- Inspire user to cook new recipes

Basic Features:
	- Search for recipes by name
	- Filter recipes
		○ Cuisine
		○ Difficulty 
		○ Ingredients
		○ Special Diet
	- User Account
	- Favorites list
	- Generate random recipe

Stretch Features:
	- User profile
	- Macros planning
	- Weekly meal planning
	- Filter by ingredient list
